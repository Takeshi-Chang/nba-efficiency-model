# NBA Shooting Efficiency Modeling

Statistical modeling project predicting NBA player true shooting percentage using interpretable player tendency metrics.

Project Overview:
- Linear regression baseline
- Polynomial feature expansion (degree 2)
- Ridge regularization
- OLS inference for coefficient interpretation

Results:
- R² improved from ~0.07 → ~0.32
- RMSE reduced from 4.61 → 3.95
